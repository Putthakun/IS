Dog Breed Neural Network Machine Learning

ข้อมูลเกี่ยวกับการพัฒนาโมเดล

การเตรียมข้อมูล: เริ่มจากการรวบรวมข้อมูลที่เกี่ยวข้องกับพันธุ์สุนัขจาก Kaggle โดยข้อมูลนี้ประกอบด้วยฟีเจอร์ต่างๆ เช่น พันธุ์, สีขน, สีตา, ความสูง, อายุขัย, ลักษณะนิสัย และปัญหาสุขภาพทั่วไป

ทฤษฎีของอัลกอริทึมที่พัฒนา: ในการพัฒนาโมเดลนี้ เราใช้ **Decision Tree Classifier** สำหรับการคาดเดาลักษณะนิสัยจากพันธุ์สุนัข ซึ่งเป็นอัลกอริธึมที่ใช้ในการแบ่งกลุ่มข้อมูลโดยการสร้างต้นไม้การตัดสินใจ

ขั้นตอนการพัฒนาโมเดลทั้ง 2 ประเภท: ในขั้นตอนแรกเราได้ทำการแปลงข้อมูลเชิงข้อความให้เป็นค่าตัวเลขโดยใช้ **LabelEncoder** และจากนั้นจึงนำข้อมูลที่แปลงแล้วมาฝึกโมเดล Decision Tree และ Neural Network ตามลำดับ

Neural Network

Neural Network (เครือข่ายประสาทเทียม): เป็นโมเดลที่ได้รับแรงบันดาลใจมาจากสมองของมนุษย์ โดยมีการเชื่อมโยงของนิวรอนหลายๆ ตัวเพื่อใช้ในการคาดการณ์หรือจำแนกข้อมูล อัลกอริธึมนี้เหมาะกับข้อมูลที่มีความซับซ้อนและไม่สามารถแยกแยะได้ด้วยการใช้ฟังก์ชันเชิงเส้นธรรมดา

Machine Learning

Machine Learning (การเรียนรู้ของเครื่อง): เป็นการใช้คอมพิวเตอร์เพื่อฝึกฝนจากข้อมูลและทำให้เครื่องสามารถเรียนรู้และทำนายผลลัพธ์ได้เองโดยไม่ต้องมีการเขียนโปรแกรมระบุไว้ล่วงหน้า

ตัวทำนายลักษณะนิสัยของพันธุ์สุนัข


ข้อมูลของ Dataset

ข้อมูลชุดนี้มาจาก Kaggle และประกอบด้วยฟีเจอร์ต่างๆ ดังนี้:

  • Country of Origin: ประเทศที่พันธุ์สุนัขนั้นๆ มาจาก
  • Breed: พันธุ์สุนัข
  • Fur Color: สีขนของสุนัข
  • Height (inches): ความสูงของสุนัข (หน่วยเป็นนิ้ว)
  • Color of Eyes: สีตาของพันธุ์สุนัข
  • Longevity (years): อายุขัยเฉลี่ยของพันธุ์สุนัข
  • Character Traits: ลักษณะนิสัยที่ทำนายจากพันธุ์สุนัข
  • Common Health Problems: ปัญหาสุขภาพที่พบได้บ่อยในพันธุ์สุนัขนั้น
